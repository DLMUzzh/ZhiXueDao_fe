# ZhiXueDao_fe
“学知岛”知识分享平台  
技术栈：前端采用: Vue、Vite（AI辅助完成）。
项目简介：知识社区 APP聚焦知识分享与互动核心场景，支持发布知识、点赞/收藏、关注/取关、首页 Feed 展示与对象存储直传，AI 生成摘要等功能。
